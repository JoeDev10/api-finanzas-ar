Autenticación

Opcional.
Gratis para empezar.

Sin API key podés hacer hasta 30 requests por minuto. Es suficiente para probar, dashboards personales o proyectos pequeños.

Para proyectos en producción con mayor throughput, enviá un header x-api-key con tu clave.

Anónimo: 30 req/min Free: 200 req/min
Header
curl https://api-finanzas-ar.vercel.app/v1/dolar \
  -H "x-api-key: tu_clave_aqui"
Query param
curl "https://api-finanzas-ar.vercel.app/v1/dolar?api_key=tu_clave_aqui"

Las claves se devuelven en el header x-ratelimit-remaining de cada respuesta.
